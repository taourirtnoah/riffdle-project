# EF Skill

## When to use
Use this skill when changing EF Core models, relationships, DbContext configuration, or migrations.

## Workflow
1. Update the domain model.
2. Add or adjust EF annotations and navigation properties.
3. Update the DbContext.
4. Create a migration.
5. Apply the migration and verify the schema.

## Example prompt
Add a new required field to `Band` and generate the EF migration steps.
