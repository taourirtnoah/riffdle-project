# Edit Form Skill

## When to use
Use this skill when creating or extending a create/edit form for data entry.

## Workflow
1. Add GET and POST actions.
2. Add a form-specific ViewModel if needed.
3. Use Razor tag helpers for inputs and validation.
4. Persist changes and redirect back to the list page.

## Example prompt
Create a playlist edit form with name, description, public flag, and owner username.
