# List Page Skill

## When to use
Use this skill when creating or extending a list/index page that shows a collection of domain objects.

## Workflow
1. Add or update the controller action.
2. Create or extend a ViewModel.
3. Build the Razor view with a table or card list.
4. Add breadcrumbs and navigation links.

## Example prompt
Create a list page for playlists that shows name, owner, visibility, and likes.
